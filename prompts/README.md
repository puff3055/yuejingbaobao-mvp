# 月经宝宝 Agent Prompt 档案

当前启用版本：[`menstrual-baby-system-v2.md`](./menstrual-baby-system-v2.md)

## 运行约定

- `server.mjs` 启动时读取当前 Prompt，不在代码里复制第二套系统提示。
- Prompt 只定义角色、语气、轮次任务、记忆与安全边界；专业事实从可追溯知识数据按本轮问题检索，不直接写死在 Prompt。
- 每轮只能完成一种任务：追问、回答、行动、进一步评估或普通对话。追问尚未回答时不得同时弹行动卡。
- 生产链路只显示通过严格结构与安全校验的联网 Agent 回复；未授权、未配置、超时、空响应或校验失败时不生成本地宝宝话术。
- 本地代码只负责发送前安全门、请求状态、结构与来源校验，以及明确标注的系统错误。
- 新版本新增文件保存，旧版本保留用于回归和追溯，不直接覆盖历史语义。

## 当前金标准

输入：`我现在小腹特别痛，下午还有会`

1. 识别疼痛与现实任务，不补造会议时间或任务类型；
2. 先询问功能影响；
3. 再确认是否与熟悉的疼痛明显不同；
4. 用户回答前不出现普通行动卡；
5. 条件足够且未触发进一步评估时，只提供一个可撤回行动；
6. 过去“没有帮助或更不舒服”的行动不能继续作为默认推荐。
